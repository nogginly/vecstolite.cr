# Vecstolite

A vector store _shard_ for Crystal with in-memory and (coming soon) file-based storage support.

### Installation

1. Add the dependency to your `shard.yml`:

```yml
dependencies:
  vecstolite:
    github: nogginly/vecstolite.cr
```

2. Run shards install
